## controls

* right click + WASD to move camera around
* scroll wheel to zoom
* select the right mask and the right color to save the clueless masked's that are in danger zones!
* quickly select the masks and colors with your numpad (1 tru 8).

Strive to survive as long as possible !!

tip: red square requires red square mask for the lil'peep to survive
